## Current project goal

Build a public Phase 1 MVP for Maryland Local Picks that helps sell and fulfill the first Catonsville card.

The Phase 1 MVP should include:

- homepage
- Catonsville edition page
- advertise page
- mock business landing page template
- mock data
- basic email opt-in placeholder
- clean mobile-first design

## Important build rule

Do not overbuild.

Do not build Phase 2+ features unless explicitly requested.

Specifically, do not add these in Phase 1 unless directly instructed:

- database-backed CRUD
- admin dashboard
- advertiser dashboard
- Stripe checkout/subscriptions
- SignalWire call tracking
- auth/login
- AI answering
- full analytics/reporting
