# Maryland Local Picks — First Edition Launch Checklist

## Page 6: Catonsville Drop #1 Action Plan

### 1. Purpose

This checklist turns the planning documents into a concrete launch sequence for the first Catonsville Local Picks edition.

The first edition should prove:

1. Businesses will pay.
2. The print + digital offer is understandable.
3. The EDDM workflow is manageable.
4. The website helps sell and fulfill the offer.
5. Advertisers are willing to renew, prepay, or continue digitally.

The first edition does not need the full future platform.

### 2. Launch Assumptions

Starting assumptions:

- Brand: Maryland Local Picks
- First edition: Catonsville Local Picks
- Domain: `marylandlocalpicks.com`
- Local project folder: `~/__code/marylandlocalpicks.com`
- GitHub repo: `maryland-local-picks`
- Card size target: 9x12
- Mailing method: USPS EDDM Retail
- Initial quantity target: approximately 2,500 households, based on actual carrier routes
- Cadence goal: every 6 weeks
- Payment model: upfront payment only for print
- Digital page: included with print placement
- First site build: public MVP only

### 3. Phase A — Business Setup

Complete before heavy selling.

- [ ] Buy or confirm control of `marylandlocalpicks.com`
- [ ] Decide whether business operates under existing entity or new entity
- [ ] Confirm payment collection method for first advertisers
- [ ] Create basic brand assets
- [ ] Create email address or alias for the project
- [ ] Create simple tracking spreadsheet or CRM pipeline
- [ ] Create folder structure for assets and proofs
- [ ] Decide first card deadline target
- [ ] Decide first mailing target window

### 4. Phase B — Website MVP

The website only needs to support sales and fulfillment for the first card.

Build:

- [ ] `/` homepage
- [ ] `/catonsville` city edition page
- [ ] `/advertise` advertiser page
- [ ] `/b/[businessSlug]` mock business landing page
- [ ] static mock businesses
- [ ] static mock offers
- [ ] static mock Pick of the Week
- [ ] placeholder newsletter opt-in
- [ ] placeholder advertiser inquiry form
- [ ] mobile-first layout
- [ ] clear CTA for residents
- [ ] clear CTA for business owners

Do not build yet:

- [ ] admin dashboard
- [ ] advertiser dashboard
- [ ] auth
- [ ] Stripe integration
- [ ] SignalWire integration
- [ ] database CRUD
- [ ] AI answering
- [ ] advanced analytics

Site launch standard:

- [ ] Looks credible enough to show a local business owner
- [ ] Explains the print + digital offer clearly
- [ ] Shows what a business landing page will look like
- [ ] Does not feel like a cheap coupon site
- [ ] Loads well on mobile
- [ ] Has no broken links
- [ ] Has no placeholder text that looks careless

### 5. Phase C — Route and Mailing Planning

Before taking print payments, confirm the real EDDM shape.

- [ ] Open USPS EDDM tool
- [ ] Search Catonsville / 21228 and nearby routes if needed
- [ ] Select candidate carrier routes
- [ ] Record route counts
- [ ] Record total estimated households
- [ ] Confirm drop-off Post Office
- [ ] Save route screenshots/notes
- [ ] Decide launch route group
- [ ] Set target print quantity based on real route total
- [ ] Confirm USPS EDDM Retail rate before collecting final payment
- [ ] Confirm size/weight eligibility with printer
- [ ] Confirm EDDM panel/indicia requirements

Important:

Do not promise an exact household count until routes are selected.

### 6. Phase D — Printing Cost Confirmation

Before final pricing or collecting many payments:

- [ ] Choose printer
- [ ] Confirm 9x12 price for target quantity
- [ ] Confirm paper stock
- [ ] Confirm turnaround time
- [ ] Confirm file requirements
- [ ] Confirm whether shipping to Maryland is included
- [ ] Confirm actual shipping cost if not included
- [ ] Confirm EDDM compatibility
- [ ] Estimate supplies cost
- [ ] Estimate total hard cost
- [ ] Determine minimum paid placements before mailing

### 7. Phase E — Placement and Pricing Decision

Define the first edition offer before selling.

- [ ] Decide available ad sizes
- [ ] Decide launch prices
- [ ] Decide whether one house/Realtor spot is included
- [ ] Decide whether one free anchor spot is included
- [ ] Decide category exclusivity rules
- [ ] Decide deadline for payment
- [ ] Decide deadline for artwork/assets
- [ ] Decide proof approval deadline
- [ ] Decide refund/cancellation policy

Suggested first-edition sizes:

- half
- single
- double
- quad

Suggested first-edition sales focus:

- single spot
- half spot
- double spot
- digital page included
- Pick of the Week as bonus or add-on

Keep the offer simple.

### 8. Phase F — Sales Materials

Prepare enough to sell, not a full corporate media kit.

- [ ] one-page rate sheet
- [ ] printed mockup/sample
- [ ] website link
- [ ] business landing page example
- [ ] payment link or invoice template
- [ ] advertiser intake form
- [ ] short walk-in pitch
- [ ] short phone pitch
- [ ] follow-up text/email
- [ ] objection responses
- [ ] list of categories still open

Must be able to answer:

- What is it?
- Where does it go?
- How many households?
- What does it cost?
- What do I get?
- When does it mail?
- How do I pay?
- What happens after the card mails?

### 9. Phase G — Prospect List

Build a focused list before calling/walking in.

Priority prospects:

- local businesses already advertising
- wrapped trucks/vans
- yard sign advertisers
- banner advertisers
- Facebook advertisers
- Google advertisers
- visible local storefronts
- home service businesses
- hungry underdog competitors
- restaurants with good offers
- coffee/pizza/ice cream/car wash anchor candidates

Track:

- business name
- category
- owner/decision maker
- phone
- email
- address
- website
- current advertising observed
- status
- next follow-up
- notes

Minimum first sales sprint:

- [ ] 50 prospects identified
- [ ] 25 high-priority prospects marked
- [ ] 10 walk-in targets chosen
- [ ] 10 phone targets chosen
- [ ] 10 email follow-ups ready

### 10. Phase H — Selling Sprint

Goal:

Sell enough paid placements to justify the first card.

Daily activity targets during launch selling:

- [ ] 5 walk-ins or owner conversations per day
- [ ] 10 calls per day
- [ ] 10 follow-up emails/texts per day
- [ ] update CRM/spreadsheet daily
- [ ] collect payment immediately when yes

Minimum before mailing:

- [ ] hard costs covered
- [ ] profit justifies time
- [ ] card has enough variety to be useful
- [ ] at least one strong household-friendly offer
- [ ] all paid placements confirmed

Do not count a spot as sold until payment is received.

### 11. Phase I — Payment Collection

For every advertiser:

- [ ] send payment link or invoice

- [ ] mark placement paid
- [ ] send thank-you/next-steps message
- [ ] send asset intake form
- [ ] set asset deadline
- [ ] set proof deadline

Rule:

No payment, no reserved print spot.

### 13. Phase K — Landing Pages and QR Codes

For each advertiser:

### 14. Phase L — Card Design

Design process:

- [ ] choose card theme
- [ ] lay out premium/locked spots

- [ ] add remaining advertisers
- [ ] add city-level QR code

- not cluttered

### 16. Phase N — Print Order

Before ordering:

- [ ] final route count confirmed
- [ ] final print quantity confirmed
- [ ] final file exported
- [ ] printer selected
- [ ] shipping confirmed
- [ ] delivery date confirmed
- [ ] file uploaded
- [ ] order confirmation saved

### 17. Phase O — EDDM Prep and Drop-Off

After cards arrive:

- [ ] inspect cards
- [ ] count/verify quantity
- [ ] separate by carrier route
- [ ] print USPS paperwork
- [ ] print facing slips
- [ ] bundle according to USPS requirements
- [ ] attach facing slips
- [ ] load vehicle
- [ ] bring payment/proof of payment
- [ ] drop at correct Post Office
- [ ] get receipt/acceptance confirmation
- [ ] save photos/receipt

### 18. Phase P — Post-Mailing Follow-Up

Within 24–48 hours after drop-off:

- [ ] notify advertisers
- [ ] send card photos
- [ ] send landing page links
- [ ] thank them
- [ ] post about edition
- [ ] begin monitoring scans/taps

Within 7–14 days:

- [ ] ask advertisers if they noticed response
- [ ] share early digital activity if available
- [ ] ask for testimonial/feedback
- [ ] start next-edition renewal conversation

Within 3–4 weeks:

- [ ] offer renewal
- [ ] offer prepaid package
- [ ] offer Pick of the Week
- [ ] offer digital-only continuation if not renewing print

### 19. First Edition Definition of Done

The first edition is complete when:

- [ ] site MVP is live
- [ ] routes are selected
- [ ] enough advertisers have paid
- [ ] card is designed
- [ ] ads are approved
- [ ] QR codes are tested
- [ ] card is printed
- [ ] EDDM is accepted by Post Office
- [ ] advertisers are notified
- [ ] landing pages remain live
- [ ] renewals have started
- [ ] lessons learned are documented

### 20. First Edition Retrospective

After the first edition, document:

- number of businesses contacted
- number of yeses
- number of noes
- close rate
- total revenue
- total hard cost
- estimated labor hours
- gross profit
- best-performing categories
- strongest objections
- most useful sales language
- QR scan count
- landing page views
- advertiser feedback
- resident feedback if any
- what to change for Drop #2

Core question:

Was this worth repeating?

If yes, improve and repeat.

If no, identify whether the issue was offer, pricing, sales activity, market, card design, or execution.
